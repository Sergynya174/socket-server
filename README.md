# Серверная часть чата

### Описание проекта: 
Чат в реальном временния для общения без регистрации.

### Технологии:
+ React.js
+ Socket.io
+ Node.js
+ Express.js

### Функционал: 

+ Обмен сообщениями
+ Отправка смайликов
+ Вход в комнату
+ Выход из комнаты
  
### Подготовка и запуск проекта

`npm i` — установка зависимостей

`npm run dev` — запускает сервер
